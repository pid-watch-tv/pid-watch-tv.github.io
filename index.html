Secure connection (HTTPS)

Welcome to the official PID Watch TV site. This page includes a basic Content Security Policy and other security-oriented meta settings to favor HTTPS connections. If you see a padlock icon in your browser address bar, it indicates an encrypted (TLS/HTTPS) connection between visitors and the hosting service.

Open site Security & Reporting

Technical note: Some security headers (for example, Strict-Transport-Security) are normally set by the server or hosting provider. GitHub Pages serves content over HTTPS; make sure Settings → Pages → Enforce HTTPS is enabled for this repository.
